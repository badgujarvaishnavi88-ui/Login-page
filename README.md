Registration Form using Python Tkinter

📌 Project Description

This project is a simple Registration Form GUI application created using Python Tkinter.
It provides a user-friendly interface where users can enter their personal details and submit the form.

The application includes different Tkinter widgets like Labels, Entry fields, Radio Buttons, Check Buttons, Option Menu, and Buttons.

🚀 Features

- User registration form interface
- Enter Full Name
- Enter Email Address
- Select Gender using Radio Buttons
- Select Country using Dropdown Menu
- Select Languages using Checkboxes
- Submit button with registration success message

🛠️ Technologies Used

- Python
- Tkinter GUI Library

▶️ How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
